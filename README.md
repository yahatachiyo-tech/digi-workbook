# 機械設計デジタルワーク

機械設計1（実教出版）第2章「機械に働く力」の学習用デジタルワークブック。

## 使い方

ブラウザで以下の URL にアクセスしてください：

**https://yahatachiyo-tech.github.io/digi-workbook/**

1. ログイン画面で出席番号（1〜40）・姓・名を入力
2. メニューから学習したい小単元を選択
3. 問題を解く

## 収録小単元

- 1-1 力とは何か（穴埋め 4問）
- 1-2 力の表し方（穴埋め 6問）
- 1-3 作用線が重なる2力の合成（作図 4問）
- 1-4 作用線が交わる2力の合成（作図 4問）
- 2-1 変位と速度（計算 5問）

## 動作環境

- Chrome / Edge / Firefox の最新版推奨
- PC・タブレット対応
- インターネット接続必須（KaTeX・フォントを CDN から取得）
